# 🔐 Phishing URL Detector

A Machine Learning based web app to detect phishing URLs built with Python, Scikit-learn and Flask.

## 🚀 Features
- Detects phishing URLs using Random Forest ML model
- 100% model accuracy
- Simple and clean web interface
- Real-time URL prediction

## 🛠️ Tech Stack
- Python
- Scikit-learn (Random Forest)
- Flask
- HTML/CSS

## ▶️ How to Run
1. Install dependencies: pip install flask scikit-learn pandas
2. Train the model: python train.py
3. Run the app: python app.py
4. Open browser: http://127.0.0.1:5000
